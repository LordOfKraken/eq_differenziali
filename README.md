# Metodi Matematici per la Fisica: Equazioni differenziali 1

Questa raccolta contiene gli appunti del corso tenuto dal professor Klemm nel primo demestre dell'anno accademico 2017/18. Sono una trascrizione fedele delle lezioni tenute dal professore, la numerazione delle equazioni e la suddivisione in capitoli è la stessa che viene fatta a lezione. 

Essendo un lavoro in corso d'opera lo stile potrebbe non essere dei migliori (scrivo il file direttamente a lezione ), eventuali miglioramenti di stile e impaginazione saranno apportati a fine corso ma cercherò di mantenere il file sempre pronto per la stampa, senza formule fuori dai bordi. 

Nel file .tex sono presenti dei commenti a separare ciò che viene fatto in ogni singola lezione, e a indicare dove il professore a disegnato delle immagini. Suddette immagini saranno caricate a fine corso, durante la revisione.

Eventuali correzioni e suggerimenti di stile sono molto ben accetti, sentitevi liberi di suggerire modifiche.
